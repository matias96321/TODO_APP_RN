# App Todo-List

> creating basics todo-lists

using async storage, building app without integration with backend of todo-lists

![](https://i.imgur.com/1nhEf9O.png)

## Installation

OS X & Linux:

```sh
yarn install
```

and

```sh
yarn start
```

choose device:

```sh
web: click w
ios emulator: press i
android emulator: press a
```
